# Agentic RAG & Workflow Automation with LangGraph

##  Description

Projet implémentant un système de **RAG Agentique** avec **LangChain** et **LangGraph**, capable de choisir automatiquement entre une base vectorielle (**ChromaDB**) et des outils externes pour répondre aux requêtes.

##  Fonctionnalités

* 🤖 RAG Agentique avec `create_react_agent`
* 🔄 Workflows avec `StateGraph`
* 🧰 Intégration de multi-outils (email, données)
* 💾 Gestion de la mémoire et historique

##  Lancement

```bash
pip install -r requirements.txt
langgraph dev
```

##  Technologies

LangChain • LangGraph • ChromaDB • Python
